# 💰 Crypto Dashboard

An interactive cryptocurrency market dashboard built in **Power BI**, tracking price trends, market performance, and key metrics across 

## 🔍 Overview

This dashboard visualizes crypto market data to help track performance and spot trends at a glance. It covers:
- [Price trends over time]
- [Market cap comparison across coins]
- [Trading volume analysis]



## 🛠️ Tools Used

- **Power BI** — dashboard design and visualization
- **[Data source]** — e.g. CoinGecko API,
## 📊 Data Source

https://api.coingecko.com/api/v3/coins/markets?vs_currency=usd&order=market_cap_desc&per_page=20&page=1

## 📁 Files

| File | Description |
|------|-------------|
| `Crypto_dashboard.pbix` | Power BI dashboard file containing all visuals and data model |

## 🚀 How to View

1. Download [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)
2. Open `Crypto_dashboard.pbix`
3. Explore the interactive charts and filters

> Note: `.pbix` files can't be previewed directly on GitHub — you'll need Power BI Desktop installed to view the dashboard.
